عن المنصة

بوابة تقنية لتبسيط اختيار النموذج والهاردوير — قابلة للتوسع

الأمان

لا نستخدم innerHTML أو حقن HTML ديناميكي لتقليل مخاطر XSS.
لا نخزن مفاتيح API أو بيانات حساسة داخل الموقع (Static-first).
عرض البيانات يتم عبر textContent فقط لزيادة الأمان.
جاهز لإضافة CSP وسياسات حماية أقوى لاحقاً.

فوائد المنصة

تساعدك تختار النموذج المناسب حسب جهازك وميزانيتك.
توضح متطلبات التشغيل بشكل عملي (VRAM/RAM) بدون تعقيد.
تختصر وقت التجارب العشوائية وتقلل الهدر في الموارد.
تقارن بين الـ GPUs لتحديد الخيار الأنسب لتشغيل النماذج محلياً.
حاسبة تقديرية لمتطلبات التشغيل حسب حجم النموذج وطريقة الضغط.

رسالة للمستخدم

هذه المنصة صُممت لتكون مرجعًا عمليًا لاختيار النماذج والعتاد بثقة. حالياً نقدم الأدوات مجاناً وبأداء عالي على GitHub Pages. مستقبلاً سنوسّع المحتوى ونضيف توصيات أعمق، وعند توفر مخزون فعلي قد نطلق متجرًا مخصصًا لتجهيزات الذكاء الاصطناعي.
